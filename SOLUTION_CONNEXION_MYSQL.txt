📝 CHANGEMENTS DANS MyConnection.java
═══════════════════════════════════════════════════════════

AVANT:
private String url = "jdbc:mysql://localhost:3306/greenledger";

APRÈS:
private String url = "jdbc:mysql://localhost:3306/greenledger?autoReconnect=true&useSSL=false&allowPublicKeyRetrieval=true";

Ces paramètres garantissent:
- autoReconnect=true → Reconnexion automatique
- useSSL=false → Pas de SSL (développement local)
- allowPublicKeyRetrieval=true → Authentification MySQL 8+
