========================================
POURQUOI CE PROBLÈME?
========================================

JavaFX charge les fichiers FXML et HTML depuis:
target/classes/

Quand vous modifiez:
src/main/resources/

Les changements ne sont PAS automatiquement copiés!

SOLUTION:
Utiliser update-resources.bat pour copier les fichiers

========================================
COMMANDES UTILES
========================================

1. Mettre à jour les ressources:
   ./update-resources.bat

2. Compiler le code Java:
   ./compile-chartjs.bat

3. Tout compiler (code + ressources):
   ./compile-all.bat

4. Lancer l'application:
   ./run.bat
