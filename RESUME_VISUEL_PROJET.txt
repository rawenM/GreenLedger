┌───────────────────────────────────────────────────────────────┐
│  🏗️  ARCHITECTURE                                             │
└───────────────────────────────────────────────────────────────┘

    📦 Model-View-Controller (MVC)
        ├─ Models/          → Entités (User, FraudDetectionResult)
        ├─ Controllers/     → Logique métier
        ├─ Services/        → Services (UserService, FraudDetection)
        ├─ DAO/             → Accès aux données
        ├─ Utils/           → Utilitaires (Email, Captcha)
        └─ Resources/       → FXML, CSS, HTML

    🔄 Flux de données
        Interface (FXML) → Controller → Service → DAO → MySQL
