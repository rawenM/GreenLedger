========================================
📁 FICHIERS CRÉÉS
========================================

1. HTML avec Chart.js:
   src/main/resources/charts/user-statistics.html
   - 4 graphiques interactifs
   - Design moderne et responsive

2. Service Java:
   src/main/java/Utils/ChartDataService.java
   - Génère les données JSON
   - Analyse les utilisateurs

3. Contrôleur JavaFX:
   src/main/java/Controllers/UserStatisticsController.java
   - Gère le WebView
   - Communication Java ↔ JavaScript

4. Interface FXML:
   src/main/resources/fxml/user_statistics.fxml
   - Page de statistiques complète

5. Script de compilation:
   compile-chartjs.bat

6. Documentation:
   GUIDE_CHARTJS_INTEGRATION.md

========================================
📊 LES 4 GRAPHIQUES
========================================

1. Répartition par Statut (Donut)
   - Actifs, En Attente, Bloqués, Suspendus

2. Répartition par Type (Pie)
   - Investisseurs, Porteurs de Projet, Admins, Experts

3. Inscriptions par Mois (Line)
   - 6 derniers mois
   - Courbe avec animation

4. Distribution Scores de Fraude (Bar)
   - 5 niveaux de risque
   - Couleurs du vert au rouge

========================================
🚀 COMMENT TESTER
========================================

1. Lancez l'application:
   ./run.bat

2. Connectez-vous en tant qu'admin

3. Dans le menu de gauche, cliquez sur:
   📊 Statistiques

4. Les graphiques s'affichent automatiquement!

========================================
🔧 TECHNIQUE
========================================

Communication Java → JavaScript:
1. Java génère JSON avec ChartDataService
2. WebView charge le HTML avec Chart.js
3. JavaScript reçoit les données
4. Chart.js affiche les graphiques

Exemple de données JSON:
{
  "status": {
    "active": 45,
    "pending": 12,
    "blocked": 3,
    "suspended": 2
  },
  "type": {
    "investor": 30,
    "projectOwner": 25,
    "admin": 5,
    "evaluator": 2
  },
  ...
}
