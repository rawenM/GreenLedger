OPTION 1: Recompiler avec compile-all.bat
------------------------------------------
1. Double-cliquez sur: compile-all.bat
2. Attendez la fin de la compilation
3. Lancez l'application avec: run.bat


OPTION 2: Tester directement (si déjà compilé)
-----------------------------------------------
1. Lancez l'application avec: run.bat
2. Testez les actions suivantes:

   TEST 1 - Connexion réussie:
   • Connectez-vous avec un compte valide
   • Vérifiez dans MySQL:
     SELECT * FROM audit_log WHERE action_type = 'USER_LOGIN' 
     ORDER BY created_at DESC LIMIT 5;

   TEST 2 - Échec de connexion:
   • Essayez un mauvais mot de passe
   • Vérifiez dans MySQL:
     SELECT * FROM audit_log WHERE action_type = 'USER_LOGIN_FAILED' 
     ORDER BY created_at DESC LIMIT 5;

   TEST 3 - Actions admin:
   • Connectez-vous en tant qu'admin
   • Validez/bloquez un utilisateur
   • Vérifiez dans MySQL:
     SELECT * FROM audit_log WHERE action_type LIKE 'ADMIN_%' 
     ORDER BY created_at DESC LIMIT 10;

   TEST 4 - Mot de passe oublié:
   • Utilisez "Mot de passe oublié"
   • Complétez avec le code à 6 chiffres
   • Vérifiez dans MySQL:
     SELECT * FROM audit_log WHERE action_type = 'USER_PASSWORD_RESET' 
     ORDER BY created_at DESC LIMIT 5;


📊 CONSULTER TOUS LES LOGS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

-- Tous les logs récents
SELECT * FROM audit_log ORDER BY created_at DESC LIMIT 20;

-- Logs par utilisateur
SELECT * FROM audit_log 
WHERE user_email = 'votre@email.com' 
ORDER BY created_at DESC;

-- Statistiques par type d'action
SELECT action_type, COUNT(*) as count 
FROM audit_log 
GROUP BY action_type;

-- Logs d'échec
SELECT * FROM audit_log 
WHERE status = 'FAILED' 
ORDER BY created_at DESC;
